# Privacy Policy for SwipeSSH

Last updated: March 13, 2026

SwipeSSH is an Android SSH client. It is designed to connect only to SSH servers explicitly selected by the user.

## Data We Access and Use

SwipeSSH may process the following data entered or created by the user:

- connection settings such as host name, IP address, port, and username
- authentication data such as passwords, private keys, and private key passphrases
- trusted host key fingerprints
- terminal input and output during an active SSH session

## How We Use This Data

SwipeSSH uses this data only to provide the app’s core SSH client functionality, including:

- connecting to SSH servers selected by the user
- authenticating the user to those servers
- storing saved connections for reuse
- verifying trusted host keys
- maintaining terminal sessions on the device while the user uses the app

SwipeSSH uses network access only for connections initiated by the user to the SSH server specified by the user.

## Data Sharing

SwipeSSH does not sell user data.

SwipeSSH does not share user data with advertising networks, analytics providers, or other third parties.

Data entered into SwipeSSH is transmitted only to the SSH server chosen by the user, as required for the app’s SSH functionality.

## Local Storage and Security

Saved connection settings are stored locally on the user’s device.

Sensitive secrets such as passwords, private keys, private key passphrases, and trusted host key fingerprints are stored locally in encrypted form using Android keystore-backed protection mechanisms.

## Data Retention and Deletion

Saved connections and related secrets remain on the device until the user deletes them or uninstalls the app.

Users can remove saved connections inside the app. Uninstalling the app removes app data, subject to normal Android behavior.

## Children

SwipeSSH is not specifically directed to children.
